# node-project
一个练习爬虫和node的小项目(尚未完成)

### 目录结构

public/ -> 公共目录 

routes/ -> 路由控制 

views/ -> 静态页面 

models/ -> 数据模型

 services/ -> 业务逻辑

view下ejs -> 引擎模板

errors -> 错误处理

### 使用方法

1.clone到本地

```git clone https://github.com/htkz/node-project.git```

2.安装依赖

 ```npm install```

3.启动

```node ./bin/www```

4.浏览器打开

打开 [http://localhost:3000/](http://localhost:3000/)
